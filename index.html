Nixie Scrolling Banner Generator

Generates an importable Factorio 2.1 blueprint for the Nixie Tubes mod. Every character gets its own alpha tube, and the complete message rotates continuously.

Lowercase becomes uppercase. Supported: A–Z, 0–9, spaces and . ? ! @ [ ] { } ( ) / * - + % :
Factorio uses 60 ticks per second.

Why the blueprint becomes large

This proven design uses one frame-selection decider for every character position in every frame. A banner of N tubes therefore uses N² deciders. It is deliberately brute-force, but it produces short, reliable neighbour-to-neighbour wiring.
